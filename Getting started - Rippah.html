Getting started

Last updated January 31, 2023

1. Downloading Rippah

The first step is to download the Rippah loader. You can do this by clicking the download button on the manage page.

2. Downloading OBS

The second step is to download OBS. OBS is required by Rippah to be able to detect pixels from your screen. OBS is free and open-source and can be downloaded from the OBS website.

3. Open loader.exe

Now it's time to open the loader. Make sure to have Valorant open and fully loaded before you open the loader. You will get prompted to enter your credentials that you used to register on the website. After you have entered your credentials, rippah will download, inject into OBS, then start.

4. In-game settings

Rippah requires some specific settings to be set in Valorant for it to be able to work. These settings are:

4.1. Raw input buffer

The raw input buffer must be set to false. This setting allows Rippah to click for you. You can find this setting under the "General" tab and then under the "Mouse" section.

4.2. Enemy highlight color

The enemy highlight color must be set to the same color as in Rippah. We recommend to use purple as it gives the best results. You can find this setting under the "General" tab and then under the "Accessibility" section.

4.3. Crosshair

The crosshair you use must have empty space in the middle. This is becuase Rippah takes the color pixels from the middle of your screen. An example of this type of crosshair is this one.

4.4. Display mode

Rippah support fullscreen, windowed fullscreen and windowed. We recommend to use windowed fullscreen because fullscreen can sometimes cause problems when tabbing out of Valorant. You can find this setting under the "Video - General" tab.

5. Web panel - config

After you have logged in, a web panel will automatically open in your default browser. This is where you will be able to configure your settings for the triggerbot. Default settings are usually good, but you might want to tweak some settings before you start using the triggerbot. Your settings is auto-saved on the cloud every minute. This means that your settings will stay the same between sessions.

5.1. Screen dimensions

In the "General" tab, make sure that your screen dimensions is correct. It automatically retreives the resolution from Valorant and should therefore be correct. You can not change this value manually. If it displays 0 as the resolution, it will usually still work.

5.2. Color

In the "Color" tab, make sure to select the color which you use in Valorant as your enemy outline color. The value in Rippah and in Valorant must be the same. We recommend to use purple as it gives the best results. The tolerance is used to configure how sensitive the triggerbot should be on the specific color. A lower tolerance means the triggerbot will be less sensitive and have less "false positives".

5.3. Presets

Different weapons in Valorant have differrent attributes. This means that the triggerbot will need different configs for different weapons. In the "Presets" tab, you can create presets for different weapons. The cooldown specifies the delay between shots. For example, a weapon with high rate of fire (ex. vandal) needs a low cooldown whilst a weapon with low rate (ex. operator) of fire needs a high cooldown. The cooldown is measured in milliseconds (1000 milliseconds = 1 second). The burst specifies how extra shots the triggerbot should shoot each time it detects an enemy. A burst of 0 means that the triggerbot only will shoot 1 shot each detection. The keybinds also make it easier to switch between different presets while in-game.

5.4. Keybinds

In the "Keybinds" tab, you can configure the keybinds for the activation/deactivation of the triggerbot. If you want a keyboard combination, press record and enter your combination. If you instead want a mouse button, use the buttons to the left. Rippah supports two different activation modes: Toggle and hold. Toggle is the default mode and will turn on the triggerbot when the keybind is pressed and then turn it off when the keybind is pressed again. Hold will turn on the triggerbot when the keybind is being hold down and then turn it off when the keybind is released. We recommend to use hold mode as it gives the least amount of false positives ex. shooting at walls with the same color as the selected outline color.

6. Using Rippah

Now everyting is setup for you to be able to start using Rippah. If you still got some questions, please contact us at our discord or email us.
